# guitar_chord_view
Guitar chord view for Android projects

# Getting Started

Add this to build.gradle

allprojects {

		repositories {
			maven { url 'https://jitpack.io' }
		}
    
	}
  
Add dependency for library

dependencies {

#unsupported yet

	        compile 'com.github.acrap:guitar_chord_view:v1.0' 
          
	}
